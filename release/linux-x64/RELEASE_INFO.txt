kctsb Release Information
==========================
Version: 3.4.2
Platform: Linux x64 (linux-x64)
Build Type: Release (LTO enabled)
Build Date: 2026-01-16 18:14 CST
Compiler: gcc (GCC) 12.2.1 20221121 (Red Hat 12.2.1-7)
glibc: 2.34 (AlmaLinux 9) (minimum requirement)

Features:
- NTL: Enabled (Number Theory Library)
- GMP: Enabled (GNU Multiple Precision)
- SEAL: Enabled (Microsoft SEAL Homomorphic Encryption)
- HElib: Enabled (IBM HElib FHE)
- LTO: Enabled (Link-Time Optimization)

Contents:
- bin/kctsb              : Command-line tool (links libkctsb_bundled.a)
- lib/libkctsb.a         : Static library (requires separate NTL/GMP/SEAL/HElib)
- lib/libkctsb_bundled.a : Bundled static library (all-in-one, like OpenSSL)
- include/kctsb_api.h    : Unified public API header

Integration:
  Option 1 (Bundled, Recommended):
    g++ myapp.cpp -I./include -L./lib -lkctsb_bundled -lz -lpthread -ldl -o myapp

  Option 2 (Separate):
    g++ myapp.cpp -I./include -L./lib -lkctsb -lntl -lgmp -lgf2x -lseal-4.1 -lhelib -lz -lpthread -o myapp
